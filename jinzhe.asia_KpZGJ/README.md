# JZ 多语言阅读助手 - Language Hub

## 产品简介

JZ 多语言阅读助手是一款功能强大的多语言学习工具，旨在帮助用户通过阅读、听力和词汇学习来提高语言能力。它支持多种文档格式，提供丰富的学习功能，为语言学习者提供全方位的学习体验。

## 核心功能

### 1. 多格式文档支持
- **PDF 文档**：支持 PDF 阅读、OCR 识别和点读模式
- **EPUB 电子书**：原生支持 EPUB 格式，提供流畅的阅读体验
- **Word 文档**：支持 DOCX 和旧版 DOC 格式
- **文本文件**：支持 TXT 纯文本文件
- **Excel 表格**：支持 XLS/XLSX 格式的词汇表

### 2. 智能 OCR 识别
- **整页 OCR**：一键识别 PDF 整页文本
- **选区 OCR**：拖拽选择区域进行精准识别
- **多语言支持**：支持中文、英语、法语、日语、韩语、泰语、阿拉伯语、印尼语、俄语、哈萨克语、越南语等多种语言
- **实时进度显示**：OCR 过程中显示识别进度

### 3. 阅读辅助功能
- **文本转语音**：支持多种语言的文字朗读
- **点读模式**：点击或选择文本区域进行朗读
- **语速调节**：可调节朗读速度
- **音量控制**：实时调整音量大小
- **多语音选择**：支持多种语音引擎和口音

### 4. 词汇学习系统
- **单词本管理**：自动收集生词，建立个人单词库
- **词汇提取**：从阅读内容中自动提取词汇
- **词汇分类**：按频率、难度等维度分类词汇
- **内置词库**：包含考研、四六级、雅思等专业词汇表
- **词汇练习**：支持词汇记忆和测试

### 5. 多语言支持
- **界面语言**：支持中文界面
- **学习语言**：支持英语、俄语等多种语言的学习
- **字体支持**：包含俄语手写体等特殊字体
- **语言检测**：自动检测文本语言

### 6. 学习工具集成
- **打字练习**：提升打字速度和准确性
- **影子跟读**：通过模仿发音提高口语能力
- **文本分段**：智能分段，便于阅读和学习
- **历史记录**：保存阅读历史和学习进度
- **数据同步**：支持本地存储和数据备份

## 技术特点

- **现代化界面**：响应式设计，支持不同设备屏幕
- **高性能**：优化的文档加载和渲染

## 快速开始

1. **启动应用**：在浏览器中打开 `index.html` 文件
2. **上传文档**：点击上传按钮，选择要阅读的文档
3. **选择语言**：在语言选择器中选择目标语言
4. **开始阅读**：使用阅读控件调整语速、音量等参数
5. **使用 OCR**：对于 PDF 文档，点击"OCR 当前页"提取文本
6. **学习词汇**：选择文本中的生词，自动添加到单词本

## 功能亮点

- **智能文本分段**：自动按句子分段，提高阅读体验
- **实时翻译**：选中单词或句子可查看翻译
- **语音合成**：高质量的文本转语音功能
- **OCR 技术**：准确识别多种语言的文本
- **词汇管理**：智能词汇提取和分类
- **多格式支持**：一站式处理各种文档格式
- **历史记录**：跟踪学习进度和阅读历史

## 应用场景

- **语言学习**：通过阅读原版书籍提高语言能力
- **考试备考**：利用内置词库备考考研、四六级、雅思等考试
- **文献阅读**：快速处理和理解外文文献
- **口语练习**：通过影子跟读和点读模式提高口语
- **词汇积累**：自动收集和管理生词
